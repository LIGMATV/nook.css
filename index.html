Nook.css

"Water.css" ahh new framework. No JavaScript. No Classes. Just raw & customizable CSS.

Inspired by the abandoned Water.css, without wasted styles and hassle!
Making an simple web page with some typography or forms should not be complicated, the idea of classless framework was firstly discussed by Lexi Mattick (@kognise) in Hacker News.
With small touch and improvement, an webpage can be really nice even without adding classes or a bit styles. Even W3C try to make a classless framework called W3C Core Styles, but what you get? Just for typography styling! Even also cheap though.

Goals

  • Responsive
  • Good code quality
  • Good browser support
  • Small size (< 2 kb)
  • Beautiful
  • No learning curve - use HTML as you would normally instead of going to the docs to learn which class does what and having to fight the framework otherwise.
  • Original Hacker News Thread

Is it responsive?

Heck yeah! It doesn't include any fancy styles so it's easily mobile responsive. Just add the famous responsive viewport tag and you'll be good to go!

In fact, try resizing this page. Everything flows super nicely as you'll see.
